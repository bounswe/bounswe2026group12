About This API

This page displays data from the JSONPlaceholder API, a free fake REST API for testing and prototyping. The API returns a sample blog post object containing:

  • ID: Unique identifier for the post
  • Title: The title of the blog post
  • Body: The content/body text of the post
  • User ID: The ID of the user who created the post
