What this API returns: The OpenHolidays API returns public holidays for a given country and date range. Each item in the response typically includes: id, startDate, endDate, type (e.g. Public), nationwide, and name (holiday name in the requested language). The data shown here is Switzerland (CH) public holidays from 1 Jan 2026 to 30 Jun 2026, with names in English (EN). It represents official days off in that country for that period.
